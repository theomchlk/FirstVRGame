# FirstVRGame

FirstVRGame est un prototype de jeu VR réalisé pour apprendre le développement VR avec Unity et enrichir mon portfolio.  
Ce projet met en avant plusieurs fonctionnalités : déplacement, interaction avec l’environnement, tir sur cible et affichage du score.

---

## Fonctionnalités

### 1. Déplacement VR
Le joueur peut se déplacer librement, se retourner et se téléporter dans l’environnement.  
[Clip des différents modes de déplacement]

### 2. Tir avec arme VR
- Utilisation d’un asset d’arme pour tirer sur des cibles.  
- Le joueur peut **grab** l’arme et tirer avec un raycast.  
- Fonctionnalité future prévue : rechargement dynamique des armes comme dans les jeux VR classiques.
